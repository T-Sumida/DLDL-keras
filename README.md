# DLDL-keras

# Under development

## Overview
keras implementation of DLDL-v2.

## Requirement
- poetry
- python=3.8
- Tensorflow==2.5.0


## Usage
```bash
$git clone git@github.com:T-Sumida/DLDL-keras.git
$cd DLDL-keras

# install library
$poetry install

# run example script
$poetry run python example.py
```
